# Builder Wizard Example Usage

This directory demonstrates using the `bldr wizard` command to interactively set up a new project.

## Basic Usage

```bash
# Navigate to your project directory
cd your-project/

# Run the wizard
bldr wizard
```

## Navigation

### Arrow Keys
- `↑` / `↓` - Navigate up/down
- `Enter` - Select current option

### Vim-Style
- `k` - Move up
- `j` - Move down
- `Enter` - Select

### Confirmation Prompts
- `Y` or `y` - Yes
- `N` or `n` - No
- `Enter` - Use default (shown in uppercase)

## What Gets Created

### Builderfile
```d
// Builderfile - Generated by 'bldr wizard'
// Language: Python
// Structure: SingleApp

target("main") {
    type: executable;
    language: python;
    sources: ["**/*.py"];
}
```

### Builderspace
```d
// Builderspace - Generated by 'bldr wizard'

workspace {
    name: "my-project";
    version: "1.0.0";

    cache {
        enabled: true;
        directory: ".builder-cache";
    }
}
```

### .builderignore
```gitignore
# Builder Ignore File - Generated by 'bldr wizard'

# Version control
.git/
.svn/

# Builder cache
.builder-cache/

# Python
venv/
.venv/
__pycache__/
*.pyc
.pytest_cache/

# IDE
.idea/
.vscode/

# OS
.DS_Store
```

## After Running the Wizard

1. **Review the generated files**
   ```bash
   cat Builderfile
   cat Builderspace
   ```

2. **Build your project**
   ```bash
   bldr build
   ```

3. **View the dependency graph**
   ```bash
   bldr graph
   ```

4. **Customize as needed**
   - Edit `Builderfile` to add dependencies or custom commands
   - Modify `.builderignore` to exclude additional files

## Tips

- The wizard scans from the current directory, so run it from your project root
- Detected languages appear first with confidence scores
- You can re-run the wizard to update configuration
- Use `Ctrl+C` to cancel at any time
